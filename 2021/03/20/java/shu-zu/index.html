什么是数组？

  • 数组(Array)，是多个相同类型数据按一定顺序排列的集合，并使用同一个名字命名，通过编号的方式对这些数据进行统一管理。
  • 数组的常见概念
    • 数组名
    • 下标（或索引）
    • 元素
    • 数组的长度
  • 数组本身属于引用数据类型的变量，而数组中的元素可以是任何数据类型，包括基本数据类型和引用数据类型。
  • 创建数组对象会在内存中开辟一整块连续的空间，而数组名中引用的是这块连续空间的首地址。
  • 数组的长度一旦确定，就不能修改。
  • 我们可以直接通过下标的方式调用指定位置的元素，速度很快。
  • 数组的分类：
    • 按照维度：一维数组、二维数组、三维数组、…
    • 按照元素的数据类型分：基本数据类型元素的数组、引用数据类型元素的数组（即对象数组）

一维数组

一维数组的使用

1.一维数组的声明和初始化

int[] id;//声明一个名为id的int类型的数组。
            //1.1 静态初始化：数组的初始化和数组元素的赋值操作同时进行
	id = new int[]{1,2,3,4};//数组是引用类型。所以初始化得new一个。
            //1.2 动态初始化：数组的初始化和数组元素的赋值操作分开进行
String[] names = new String[5];
//总结：数组一旦初始化完成，其长度就确定了

2.如何调用数组指定位置的元素

//2.如何调用数组指定位置的元素：通过角标的方式调用
//数组的角标是从0开始的，到数组的长度-1结束
names[0] = "MiSaKa";
names[1] = "KiRuTo";
names[2] = "ZhangSan";
names[3] = "SaKuRa";
names[4] = "YaMeDe";

3.如何获取数组的长度

//3.如何获取数组的长度：属性：length
//比如我想看看names[]这个数组有多长（有几个元素）
System.out.println(names.length);

4.如何遍历数组

//4.如何遍历数组
//数组中，主要变化的就是下标。于是，使用for循环
for(int i=0;i< names.length;i++){
    System.out.println(names[i]);
}

6.数组的内存解析

有如下代码,请画出简易内存解析图：

int[] arr = new int[]{1,2,3};
String[] arr1 = new String[4];
arr1[0] = "ZhangSan";
arr2[1] = "LiSi";
arr1 = new String[3];

解释：

首先我们数组的这些代码都是放在main方法的，而方法中的变量都是局部变量，局部变量都是放在栈空间中的。

于是int[] arr这一句，会在栈空间中开辟出一块地方，放arr这个变量。

而堆空间主要是放new出来的对象、数组等。那么new int[]{1,2,3}这句，因为这个数组有三个元素，那么会在堆空间中开辟三个连续的地方，并且，这三个地方默认初始化为0。又因为我们这是静态数组，在new的时候已经把元素的值写上了，那么这三个空间在开辟出来的同时，里面的初始化值”0“会立马被覆盖掉，变成1，2，3。

在堆空间开辟的时候，会有个首地址值，这个首地址值会赋给栈空间中的变量arr，然后栈空间中的变量就可以根据首地址值指向堆空间中具体的元素。

二维数组

一个一维数组，如果其元素还是一维数组，则整体为二维数组。

因为数组虽然本身是引用类型，但他的元素可以是基本数据类型，也可以是引用数据类型，而数组也是引用数据类型，所以元素也可以是数组。

其实，从数组底层的运行机制来看，其实没有什么多维数组。

1.二维数组的声明和初始化

//1.二维数组的声明和初始化
//静态初始化
int[][] arr = new int[][]{{1,2,3},{4,5},{6,7,8}};
//动态初始化1
String[][] arr2 = new String[3][2];//数组长度为3，每个元素是一个数组，元素数组的长度是2
//动态初始化2
String[][] arr3 = new String[3][];

//也是正确的写法
int[] arr4[] = new int[][]{{1,2,3},{4,5},{6,7,8}};
int[] arr5[] = {{1,2,3},{4,2.5},{6,7,8}};

2.如何调用数组指定位置的元素

//2.如何调用数组指定位置的元素
System.out.println(arr[0][1]);

System.out.println(arr2[0][1]);

3.如何获取数组的长度

System.out.println(arr4.length);//判断数组长度
System.out.println(arr4[1].length);//因为元素也是数组，当然也能判断长度

4.如何遍历二维数组

//4.如何遍历二维数组 需要用循环嵌套
for(int i=0;i<arr4.length;i++){
    for (int j=0;j<arr4[i].length;j++){
        System.out.print(arr[i][j]);//输出第i个元素的第j个数组
    }
    System.out.println();
}

5.数组元素的默认初始化值

/*规定：二维数组分为外层数组的元素与内层数组的元素
5.数组元素的默认初始化值
int[][] arr6 =new int[4][3];
外层元素：arr[0],arr[1]等
内层元素：arr[0][0],arr[1][2]等
 */

int[][] arr6 = new int[4][3];
System.out.println(arr6[0]);//外层数组默认初始化值是地址
System.out.println(arr6[0][0]);//内层数组由于已经精确到了某一位，所以默认初始化值是基本数据类型的默认初始化值。

/*针对于初始化方式一：比如：int[][] arr = new int[4][3]
外层元素的初始化值为：地址值
内层元素的初始化值为：与一维数组初始化情况相同

针对初始化方式二：比如：int[][] arr = new int[4][];
外层元素的初始化值为：null
内层元素的初始化值为：不能调用，否则报错。因为根本没定义。*/
